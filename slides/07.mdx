# Pitfall 4: Overthinking performance

Does this worry you?

```jsx
function YoYoScreen() {
  const [yoyo, dispatch] = React.useContext(YoYoContext)

  const handleUpdate = updates => dispatch({type: 'update', updates})

  return <YoYoInfo yoyo={yoyo} onUpdate={handleUpdate} />
}
```

Is this better?

```jsx
function YoYoScreen() {
  const [yoyo, dispatch] = React.useContext(YoYoContext)

  const handleUpdate = React.useCallback(
    updates => dispatch({type: 'update', updates}),
    [dispatch],
  )

  return <YoYoInfo yoyo={data} onUpdate={handleUpdate} />
}
```

Maybe it is, maybe it isn't...

**Takeaway**: Be considerate of performance, but also be considerate of your
code complexity. _Measure first._
